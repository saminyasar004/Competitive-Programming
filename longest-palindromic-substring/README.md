# **Longest palindromic substring**

Given a string `s`, return the longest palindromic substring in `s`.

### **Example 1:**

```javascript
Input: s = "babad";
Output: "bab";
```

**Note: `"aba"` is also a valid answer.**

### **Example 2:**

```javascript
Input: s = "cbbd";
Output: "bb";
```

### **Example 3:**

```javascript
Input: s = "a";
Output: "a";
```

### **Example 4:**

```javascript
Input: s = "ac";
Output: "a";
```
